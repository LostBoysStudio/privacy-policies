Eggs and Bakey Privacy Policy

Last updated: February 2024

Data Collection and Use

  • Heart rate data collected via HealthKit
  • Used only for drowsiness detection
  • Processed locally on device
  • No data stored long-term
  • No data shared with third parties

Required Permissions

  • Heart rate access via HealthKit
  • Notification permissions
  • Background processing capabilities:
    • Audio playback
    • Remote notifications
    • Mindfulness tracking
    • Workout processing
  • Audio permissions for alerts

Data Protection

  • Local processing only
  • No cloud storage
  • No data sharing
  • Session-based monitoring

Background Processing

  • The app processes data in the background for continuous monitoring
  • Audio alerts can play in the background
  • Remote notifications keep you informed even when the app isn't active
  • Mindfulness and workout data processing occurs locally
